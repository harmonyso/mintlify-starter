## About Slack

Slack is a team collaboration platform that brings together messaging, file sharing, and integrations in one unified workspace. Connecting Slack to Harmony transforms your channels into an intelligent IT helpdesk with autonomous ticket resolution, AI-powered responses, and seamless workflow automation.

## What the Slack integration enables

| Capability           | Description                                                                                                  |
| -------------------- | ------------------------------------------------------------------------------------------------------------ |
| Autonomous Help Desk | Transform Slack channels or DMs into an AI-powered help desk with autonomous ticket resolution               |
| Team Inbox           | Centralize all IT tickets in a dedicated channel for support agents to triage and collaborate                |
| Automation workflows | Trigger workflows, receive notifications, and execute IT actions directly from Slack                         |
| Slash Commands       | Use `/harmony` from any channel or DM to instantly create tickets and execute commands                       |
| Message Actions      | Convert any existing message into a ticket using the message context menu                                    |
| AI Responses         | Get intelligent answers from Harmony's AI trained on your knowledge base and IT procedures                   |

## Prerequisites

Before connecting Slack to Harmony, ensure you have:

1. **Slack Workspace Admin access** or permissions to install applications
2. **A Harmony account** with admin access to manage integrations
3. **At least one Slack channel** where you want Harmony to operate

<Note>
  Standard Slack workspace members can install apps, but your organization's app approval policy may require admin approval.
</Note>

## Connect Slack to Harmony

### Step 1: Initiate the Connection

<Steps>
  <Step title="Navigate to Integrations">
    1. Log into your Harmony dashboard
    2. Click **Settings** in the sidebar
    3. Click **Integrations** in the settings menu
    4. Click **Slack** under the **Collaboration** section to begin the OAuth flow

    <img src="/images/screenshots/slack-integrations-page.png" alt="Navigate to Slack integration in Harmony dashboard" />
  </Step>

  <Step title="Authorize in Slack">
    You'll be redirected to Slack's authorization page.

    **Important**: Ensure you have the correct Slack workspace selected in the top-right corner before proceeding.

    1. Review the requested permissions (see [Permissions](#permissions-explained) below)
    2. Select a channel to connect to Harmony initially
    3. Click **Allow** to authorize the connection

    <img src="/images/screenshots/slack-authorization-page.png" alt="Click Allow to authorize Harmony in Slack" />

    At this point, Slack should be successfully connected to Harmony.
  </Step>
</Steps>

## Configure Slack as Your IT Help Desk

After connecting Slack, you'll land on the Slack configuration page in Harmony where you can configure how Harmony operates in your channels.

### Configure Your Connected Channel

Once you've connected Slack, configure how Harmony should operate in the channel:

<Steps>
  <Step title="Enable the Channel">
    1. Click **Settings** in the sidebar
    2. Navigate to **Desks** in the settings menu
    3. Select your desk and click **Notifications**
    4. Click **Add destination** and select **Slack**
    5. Configure the notification events and select your Slack channel

    <img src="/images/screenshots/slack-enable-channel.png" alt="Enable and configure Slack channel in Harmony" />
  </Step>

  <Step title="Test the Integration">
    1. Go to your Slack channel
    2. Type a test message like "Help with password reset"
    3. Harmony should respond to your request
  </Step>
</Steps>

### Connect Additional Channels

You can connect more channels in two ways:

**Method 1: Through notification settings**
1. Navigate to **Settings → Desks → Notifications**
2. Click **Add destination** and select **Slack**
3. Choose a different channel from the dropdown
4. Configure the notification events for the new channel

**Method 2: Using @mention in Slack**
1. Go to any Slack channel where you want to add Harmony
2. Type `@Harmony` to mention the app
3. Click the **Add them** button when prompted
4. Configure the notification settings in Harmony

## Interacting with Harmony in Slack

Harmony provides multiple ways to interact directly from Slack:

### Slash Commands

Use `/harmony` from any channel or DM to create tickets instantly:

```
/harmony Password reset needed for john@company.com
```

### Message Actions

Convert any existing message into a ticket:

1. Hover over any message
2. Click the **⋯** (more actions) menu
3. Select **Create Harmony Ticket**

### @Mentions

Tag `@Harmony` in channels to activate assistance:

```
@Harmony How do I reset a user's MFA?
```

### Thread Replies

Continue natural conversations with context preservation—just reply in the thread where Harmony responded.

## Permissions Explained

Harmony requests the following Slack permissions:

| Permission              | Why we need it                                              |
| ----------------------- | ----------------------------------------------------------- |
| `chat:write`            | To post messages and responses in channels and DMs          |
| `channels:read`         | To list and access channels where Harmony is added          |
| `channels:history`      | To read message history for context and ticket creation     |
| `users:read`            | To identify users and personalize responses                 |
| `commands`              | To enable `/harmony` slash commands                         |
| `im:write`              | To send direct messages to users                            |
| `im:history`            | To read DM history for ticket context                       |
| `reactions:write`       | To add reaction emojis for ticket status (✅ resolved, etc.) |
| `files:read`            | To access attachments in ticket requests                    |

## Troubleshooting

### Harmony doesn't respond to messages

**Possible causes:**
- You need to @mention Harmony in the channel
- Harmony lacks permissions in the channel

**Solutions:**
1. Ensure you're @mentioning Harmony in the channel
2. Re-invite Harmony to the channel: Type `@Harmony` and click **Add them**

### Tickets aren't being created

**Possible causes:**
- Messages don't meet ticket creation criteria
- Integration connection expired

**Solutions:**
1. Try explicitly creating a ticket with `/harmony [your request]`
2. Reconnect the integration: **Settings → Integrations → Slack → Reconnect**
